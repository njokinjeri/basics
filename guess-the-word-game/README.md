# Guess The Word Game

![Design preview guess the word game challenge](./design/preview.jpg)

### challenge

Users should be able to:

    - View a randomly generated scrambled word when the page loads and whenever the Random button is selected.
    - Input guesses one letter at a time, with focus automatically advancing to the next available input after each entry.
    - See real-time feedback showing: 
            - The total number of incorrect attempts (tries).
            - Which letters have been entered incorrectly (mistakes).

    - Generate a new scrambled word at any time by selecting the Random button.
    - Fully reset the game state—including all inputs, mistakes, and tries—by selecting the Reset button.
    - Have the game automatically reset when the number of incorrect tries or mistakes reaches six (6).
