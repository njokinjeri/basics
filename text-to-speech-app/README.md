# Basic Text To Speech

![Thumbnail for the Basic Text To Speech coding challenge](./assets/thumbnail.jpg)

### Challenge 

Users should be able to:

    - Enter text in a textarea to be converted into speech.
    - Select a voice from available options in the dropdown menu.
    - Play the text as audio by pressing the “Text to Speech” button.
    - View the optimal layout for the app depending on their device's screen size
    - See hover states for all interactive elements on the page
